# 🧮 Simple Calculator

This is a simple calculator that I built with HTML, CSS, and JavaScript 
to practice the basics of DOM manipulation and event handling.

---

## ✨ Features

- ➕➖✖️➗ Basic operations: addition, subtraction, multiplication, division
- 🔢 Decimal point (only one allowed per number)
- 🧹 Clear button (C) to reset everything
- ⌫ Backspace button to delete the last digit
- ⚠️ Error handling for division by zero ("Error")

---

## 🧰 Technologies Used

- 📄 HTML5
- 🎨 CSS3 (Grid Layout)
- ⚡ JavaScript (Vanilla, no framework)

---

## 📚 What I Learned

- 🔧 Working with JavaScript variables and functions
- 🖱️ onclick for button clicks
- 🔀 Simple logic with if/else and switch statements
- 🔢 parseFloat() to convert text into numbers
- 🧩 CSS Grid for arranging the buttons

---

## 📂 Project Structure

```
simple-calculator/
├── index.html   (structure)
├── style.css    (design)
└── script.js    (logic)
```

---

## 🚀 How to Run

Just open the `index.html` file in your browser. No installation needed! 🎉
